Conversion tracking for audio campaigns, which use the Validate attribution platform.

== Description ==

Provides a simple method for enabling conversion tracking for audio campaigns. The plugin automatically loads a Validate Javascript tag into the header of the site. The tag loads asynchronously, to prevent blocking other elements.
